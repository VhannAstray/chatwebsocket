# chatwebsocket
Websocket du chat 
Serveur Websocket nodeJS + Express

In order to install :

$: cd chatwebsocket
$: npm init
$: npm i ws express

Check if typescript package is already installed
$: npm list typescript

if not :
$: npm i typescript -g

install express and node @types :
$: npm i @types/ws @types/express -D

You're ready to ROCK !
